# Pirate Map Chatbot

A small browser-based AI-style chatbot with a pirate treasure map theme. It runs fully in the browser and uses a simple response engine to answer like a friendly navigator reading clues at sea.

## Files

- `index.html` - chatbot page
- `style.css` - interface styling
- `script.js` - chat behavior and themed responses

## Run

Open `index.html` in a browser.

## Try asking

- `hello`
- `read the map`
- `where is treasure?`
- `storm warning`
- `help me navigate`
